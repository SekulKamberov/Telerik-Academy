# Course Evaluation * **Practical exam** - 55% * **Practical exam peer review** - 10% * **Homework** - 5% * **Team work** - 15% * **Attendance** - 5% * **Forum activity** - 5% * **Helping other students** - 5%
# Practical Exam * Your task will be to create a Single-page application (SPA) by a given REST API writen in Node.js * Create a client running in the browser * Fetch data from the REST API and visualize it * Send data to the REST API * Validate input and handle errors * Unit-test specific components from the app using Mocha and Chai
# Teamwork * The teamwork assignment is a way to **join forces** with other colleagues to create a **more complete web application**: * Team members are randomly selected * Based on score from previous courses * Each active team member **wins the score of the whole team** * Unactive team members win a negative bonus of -5 * The assignment can be found at [our github](http://github.com/TelerikAcademy/JavaScript-Applications/blob/master/Teamwork/README.md) * Each teamwork project must be uploaded to our [Showcase System](http://best.telerikacademy.com/)
